Couper une vidéo et extraire une scène d'un film

Comme promis dans le dernier billet, voici comment on fait pour extraire une scène d'un film, par exemple pour en faire un fichier .gif (cf post précédent)... Pour cela vous utiliserez mencoder :

sudo apt-get install mencoder

Et la syntaxe est la suivante :

mencoder -ss     h:mm:ss    \
         -endpos ...
